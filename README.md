Project Overview

The project uses Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn to clean, analyze, and visualize Diwali sales data.
The analysis answers key business questions including:

Which customer segments purchase the most?

Which product categories have the highest sales?

Which states contribute the most to orders and total amount?

What are the top-selling products?

Dataset Used

File name: Diwali Sales Data.csv

A retail e-commerce sales dataset containing customer demographics, product details, and order information.
